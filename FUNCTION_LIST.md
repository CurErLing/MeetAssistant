
# 积木会议助手 - 详细功能清单

## 功能概览 (Feature Overview)

| 核心模块 | 功能名称 | 功能描述 | 对应 UI/入口 |
| :--- | :--- | :--- | :--- |

| | **文件导入** | 支持上传本地音频文件 (MP3, WAV, M4A)，自动提取元数据。 | 主页/列表页 -> "导入文件" |
| | **硬件同步** | 通过 Web Bluetooth 连接 MeetingMic Pro 硬件，批量同步录音文件。 | 主页/列表页 -> "同步设备" |

| | **角色分离** | 自动识别不同发言人 (Speaker Diarization)，支持手动修正与声纹关联。 | 会议详情页 -> 转写列表 |
| | **智能分析** | 基于预设或自定义模板 (如纪要、待办、决策) 生成结构化总结。 | 会议详情页 -> "+" 添加分析视图 |

| | **内容修订** | 支持对转写结果进行逐句编辑、复制及发言人姓名修改。 | 会议详情页 -> 转写气泡 |

| | **热词优化** | 配置专属词汇 (如公司名、术语)，提升 AI 转写的识别准确率。 | 侧边栏 -> 热词管理 |
| | **模板中心** | 管理 AI 分析模板，支持自定义 Prompt 指令以适应不同会议场景。 | 侧边栏 -> 模板管理 |

| | **回收站** | 存放已删除的会议，支持一键还原或彻底删除 (7天自动清理)。 | 侧边栏 -> 回收站 |

| | **团队空间** | 支持绑定 Team ID，切换个人/团队视图，实现数据隔离与共享。 | 侧边栏 -> 个人头像 -> 个人中心 |

## 详细功能说明

### 1. 会议采集 (Acquisition)
*   **多模式支持**: 无论是临时产生的灵感 (在线录音)、存量的历史文件 (导入)，还是专业设备的录制 (硬件同步)，均可统一管理。
*   **硬件同步协议**: 实现了自定义 BLE 协议，支持读取设备电量、存储空间、文件列表，并支持断点续传和批量下载。
*   **预处理**: 上传前支持简单的音频裁剪，去除无效的首尾静音或无关内容。

### 2. 智能转写与分析 (Intelligence)
*   **Gemini 驱动**: 底层接入 Google Gemini Multimodal 模型，直接处理音频数据，无需中间转换，延迟更低，理解力更强。
*   **动态模板**: 
    *   **通用**: 标准摘要。
    *   **会议**: 详细纪要、待办事项 (Action Items)、关键决策。
    *   **面试**: 候选人评估报告。
    *   **自定义**: 用户可编写 Prompt 创建专属分析视角。

### 3. 声纹与角色识别 (Identity)
*   **未知发言人**: 系统默认标记为 "Speaker X"。
*   **手动修正**: 用户可手动修改某一句的发言人，系统会自动更新该 ID 对应的所有片段。
*   **声纹注册**: 
    *   **主动录入**: 朗读一段提示文本录入声纹。
    *   **被动提取**: 从现有会议录音中提取某段音频作为声纹样本。
*   **自动关联**: 再次上传会议时，系统匹配声纹库，自动标记真实姓名。

### 4. 数据组织与安全 (Organization)
*   **层级管理**: 扁平化的文件列表 + 灵活的文件夹分类。
*   **回收机制**: 软删除机制，提供“后悔药”，防止误操作导致数据丢失。
*   **团队隔离**: 基于 `Team ID` 的逻辑隔离，确保不同团队的数据互不可见 (逻辑层实现)。

### 5. 交互体验 (UX)
*   **播放器联动**: 点击转写文字，音频自动跳转到对应时间；拖动音频进度条，转写列表自动滚动跟随。
*   **响应式布局**: 完美适配桌面端 (Table视图) 和移动端 (Card视图)，侧边栏在移动端自动转为抽屉式导航。
*   **乐观 UI**: 所有增删改操作界面即时响应，后台异步同步数据，提供流畅的操作体验。
